servings: 2

recipe_image: ""
ingredients:
  - section: "Base commune (à préparer ×2)"
    items:
      - qty: 200
        unit: ml
        item: "eau"
      - qty: 80
        unit: g
        item: "sucre"
      - qty: 100
        unit: ml
        item: "jus de citron vert"
      - qty: 1
        unit: pincée
        item: "sel"
  - section: "L'asymétrie CP"
    items:
      - qty: 1
        unit: goutte
        item: "vodka (pour la base B seulement)"
directions:
  - "Préparer la base commune : mélanger l'eau, le sucre, le jus de citron et le sel. Remuer jusqu'à dissolution complète."
  - "Diviser en deux récipients identiques (Base A et Base B). Visuellement, elles sont indiscernables."
  - "Ajouter **1 seule goutte de vodka** dans la Base B. C'est la « violation de CP » — le petit déséquilibre qui change tout."
  - "Verser chaque base dans un bac plat (type plaque à glaçons ou plat rectangulaire) et placer au congélateur."
  - "Au bout d'1 h, gratter les cristaux avec une fourchette pour former le granité. Recommencer toutes les 30 min pendant 3 h."
  - "Sortir les deux bacs et laisser reposer 3 min à température ambiante, côte à côte."
  - "Observer : la Base B (avec la vodka) fond visiblement plus vite — l'alcool abaisse le point de congélation d'environ 0,5°C. La Base A reste structurée."
  - "Si on mélange les deux, le goût penche du côté A. La matière a survécu."
notes: "**Le parallèle avec l'Univers** : deux bases nées identiques dans le même congélateur (Big Bang). Une asymétrie minuscule (1 goutte pour ~380 g de liquide, soit ~1/7500) change toute la suite. La Base B (antimatière) disparaît — la Base A (matière) reste. Pourtant, personne ne peut dire en les regardant au départ laquelle va gagner. C'est exactement le problème de l'asymétrie baryonique : les lois qu'on connaît ne suffisent pas à prédire pourquoi la matière a gagné."
